Usage

1. Download the "anchor.less" file

2. Reference for use globally with: 

@import "anchor.less";

3. Use your favorite less compiler or try Prepros (it's FREE and open-source)

Example:

#foo {
   .back(#222, '../images/logo.png', no-repeat, left top);
   .border(0 1px 1px 0, solid, #ccc);
   .rounded(5px);
   .width(100%);
   .width(50%, 640px);
   .width(20%, 960px);
}

Compiles to:

#foo {
   background: #222222 url('../images/logo.png') no-repeat left top;
   border-width: 0 1px 1px 0;
   border-style: solid;
   border-color: #ccc;
   -webkit-border-radius: 5px;
   -moz-border-radius: 5px;
   border-radius: 5px;
   width: 100%;
}@media all and (min-width: 640px) {
    #foo {
        width: 50%;
    }
}@media all and (min-width: 960px) {
   #foo {
       width: 20%;
   }
}
